# Pavani Studio

A premium, modern web presence for Pavani Studio.

## Features
- **Modern UI/UX**: Clean, professional design with smooth transitions.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Vercel Optimized**: Ready for instant deployment with clean URLs.

## Project Structure
- `index.html`: Home page
- `about.html`: About the studio
- `services.html`: Services offered
- `portfolio.html`: Showcase of work
- `contact.html`: Contact information
- `css/style.css`: Main styling
- `js/script.js`: Interactive elements

## Deployment

### GitHub
1. Create a new repository on GitHub.
2. Initialize and push the code:
   ```bash
   git add .
   git commit -m "Initial commit: Ready for hosting"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/pavanistudio.git
   git push -u origin main
   ```

### Vercel
1. Go to [vercel.com](https://vercel.com).
2. Click **Add New** > **Project**.
3. Import your GitHub repository.
4. Vercel will automatically detect the static site and deploy it.

## Local Development
Simply open `index.html` in your browser or use a local server like Live Server (VS Code extension).
